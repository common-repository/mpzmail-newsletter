== Description ==

The MPZMail plugin enables you to easily add a newsletter signup form for your MPZMail group on your WordPress site.

You will need a free MPZMail account before your begin which can be registered at http://www.mpzmail.com.

== Installation ==

This section describes how to install the plugin and get started using it.

= Version 2.8+ =
1. Unzip our archive and upload the entire `mpzmail` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and look for "MPZMail" in the menu
4. Enter your MPZMail userID and group ID. Instructions on how to obtain these are on the page.
8. Finally, go to Appearance->Widgets and drag the `MPZMail Newsletter Signup Form` widget into one of your Widget Areas
9. And you are DONE!
